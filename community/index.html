Join the APE Community

Whether you're an experienced developer or an end-user just getting started with APE, there are many ways to participate in the APE community. Take a few minutes to discuss APE with other users and become an active member of the APE community.

Google Groups

The APE Google Group is the rally point of the community. Share your technical knowledge about APE with the community or receive help whenever you need it. The community is always there to help you with APE related questions.

GitHub

The APE Project is 100 % Open Source. The source code for both APE Server and the Javascript Client Framework can be found on GitHub.

Get involved

Documentation

Help produce official documentation, fix English mistakes on the wiki, translate existing documentation into other languages, share the solution to a problem, check documents for accuracy

Development

Package APE for various Linux distributions (We are looking for Debian, Fedora, Red Hat, and Mandriva packagers), port APE to windows, write more APE modules and share them with the community, build cool applications using APE

Bugs

Help us to improve APE by testing alpha and beta releases on Github. Report bugs on our bugtracker

Support

Share your technical know-how and help other users by joining our Google group

Who uses APE Project?

The APE Project is in active use at several environments.

Cool APE related projects

  • ApePubSub - A PubSub JavaScript Framework for the APE Server
  • APE-Planet - A method to synchronize data and UI elements between users
  • APE iOS Client - A client to connect to APE Server from iOS devices
